Histogrammes et boîtes à moustaches pour les distributions continues

  1. Choisissez l’une des variables continues de l’ensemble de données, telles que age ou age_premiere_naissance. Ensuite, générez un histogramme simple de cette variable. Assurez-vous de fournir le titre, les étiquettes, etc. nécessaires pour mieux communiquer les informations au public.
# Histogramme simple en forçant la conversion numérique
ggplot(ir_propre, aes(x = as.numeric(age))) +
  geom_histogram(binwidth = 2, fill = "steelblue", color = "white") +
  labs(title = "Distribution des âges des répondantes",
       x = "Âge (en années)", 
       y = "Nombre de femmes")

  1. Maintenant, créez une boîte à moustaches pour montrer la distribution de la variable continue que vous avez choisie précédemment. La distribution doit être regroupée par l’une des variables liées aux connaissances sur le paludisme. Incorporez l’esthétique fill pour colorer les boîtes. Ajustez également les couleurs, les étiquettes, les titres, etc. comme bon vous semble.
# Boîte à moustaches comparant les groupes


ggplot(ir_propre, aes(x = paludisme_peut_tuer, y = age, fill = paludisme_peut_tuer)) +

Diagrammes à barres pour les distributions discrètes

  1. Sélectionnez l’une des variables catégorielles de l’ensemble de données. Poursuivez en générant un simple diagramme à barres pour afficher la distribution de cette variable.
# Diagramme à barres simple

ggplot(ir_propre, aes(x = niveau_education)) +
  geom_bar(fill = "darkorange") +
  labs(title = "Distribution du niveau d'éducation",
       x = "Niveau d'éducation",
       y = "Nombre de femmes")

  1. Ensuite, adaptez votre graphique ci-dessus pour créer un diagramme à barres empilées en pourcentage. Pour ce faire, ajoutez l’esthétique fill et définissez-la sur une variable de connaissances sur le paludisme de votre choix ; par exemple : aes(fill = VAR_PALUDISME). N’oubliez pas d’ajouter l’argument position approprié. Le graphique final devrait comparer comment les réponses à la question de l’enquête sur les connaissances sur le paludisme varient entre les groupes.
# Diagramme à barres empilées en pourcentage

ggplot(ir_propre, aes(x = niveau_education, fill = paludisme_peut_tuer)) +
  geom_bar(position = "fill") +
  scale_y_continuous(labels = scales::percent) +
  labs(title = "Perception du paludisme selon le niveau d'éducation",
       x = "Niveau d'éducation",
       y = "Proportion")

  1. Enfin, créez un diagramme à barres GROUPÉ avec deux variables catégorielles de votre choix. Ajustez les couleurs, les étiquettes, les titres, etc. comme bon vous semble.
# Diagramme à barres groupé

ggplot(ir_propre, aes(x = urbain_rural, fill = paludisme_peut_etre_gueri)) +
  geom_bar(position = "dodge") +
  labs(title = "Réponses : Le paludisme peut être guéri",
       x = "Lieu de résidence",
       y = "Nombre de femmes",
       fill = "Réponse")

Défi bonus : Diagrammes en donut

Représentez les données de votre diagramme à barres groupé sous forme de séries de diagrammes en donut.

INDICE : La fonction facet_wrap() de {ggplot2} sera utile.

ir_propre %>%
  count(urbain_rural, paludisme_peut_etre_gueri) %>%
  group_by(urbain_rural) %>%
  mutate(pct = n / sum(n),
         pos = cumsum(pct) - pct/2) %>%
  ggplot(aes(x = 2, y = pct, fill = paludisme_peut_etre_gueri)) +
  geom_col(width = 1, color = "white") +
  coord_polar(theta = "y") +
  facet_wrap(~urbain_rural) +
  xlim(0.5, 2.5) +
  theme_void() +
  labs(title = "Réponses sur la guérison du paludisme (donut par résidence)")
